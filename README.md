# Wildberries Parser

Профессиональный парсер каталога товаров Wildberries по поисковому запросу с выгрузкой результатов в XLSX (Excel). Не зависит от громоздкой бэкенд-архитектуры и решает задачу сбора данных с помощью легковесного и надежного скрипта на Python.

## Ключевые особенности (Senior Solutions)

- **Обход антибот-защиты (Cloudflare / Qrator):** Вместо `requests` используется `curl_cffi`, полностью подменяющий TLS-отпечатки на Google Chrome. Это позволяет собирать данные без риска блокировки и ошибки `429 Too Many Requests`.
- **Многопоточная архитектура (Обход GIL):** Обычные библиотеки вроде `curl_cffi` блокируют GIL при массовых загрузках. Мы перевели скачивание карточек и фото на внутренний пакет Python (`urllib`), что дает настоящую, нативную многопоточность. Полный сбор данных 1000 товаров (API v4, seller, img) завершается за **40-60 секунд**.
- **Динамический резолв Basket-серверов:** Скрипт интеллектуально вычисляет и кэширует правильные хосты (`basket-01` ... `basket-52` и далее), на которых лежат фотографии и описания конкретных товаров. Это гарантирует отсутствие ошибок 404 для самых новых товаров.
- **Поддержка актуального API v18:** Реализована логика работы напрямую с корневым массивом `products` (v18), а также пакетная загрузка (`batch-requests`) до 100 товаров за раз (`v4/detail`).

## Результат работы

После выполнения в текущей директории появятся два файла:

1. `wildberries_catalog.xlsx` — полный каталог всех найденных товаров.
2. `wildberries_filtered.xlsx` — выборка товаров по условию: рейтинг ≥ 4.5, стоимость ≤ 10 000 руб, страна производства: Россия.

## Требования

- Python 3.10+
- Подключение к интернету

## Установка и запуск

```bash
# Установка зависимостей (curl_cffi, openpyxl)
pip install -r requirements.txt

# Запуск парсера
python main.py
```

## Собираемые данные (Структура колонок)

Скрипт формирует таблицу со строгим соответствием требованиям:

| Колонка в XLSX | Описание |
|---|---|
| Ссылка на товар | Формируется на основе ID (nm_id) |
| Артикул | Идентификатор товара |
| Название | Наименование товара |
| Цена | Актуальная цена со скидкой (в руб) |
| Описание | Полный текст описания из карточки |
| Ссылки на изображения через запятую | Полные URL всех фотографий |
| Все характеристики с сохранением их структуры | Склейка вида `Группа: Свойство — Значение` |
| Название селлера | Название ИП / ООО или бренда |
| Ссылка на селлера | Формируется на основе `supplierId` |
| Размеры товара через запятую | Доступные размеры |
| Остатки по товару (число) | Сумма остатков всех размеров на складах |
| Рейтинг | Средний рейтинг товара |
| Количество отзывов | Общее число отзывов покупателей |
